Week 5 - Week 6

Hardware Coding

As the server was already done the next step was to connect node MCU to the server. The protocol that used for this connection is MQTT. Having done the research about this protocol our group wrote the code for node MCU to connect at first to wi-fi, then to the server. The code also was able to create an avatar on the server, publish the data on it and check if any data was present on specific pin of this avatar and read it. Flowless functionality of this code was crucial and after completing this task we were able to connect sensors to the ESP module and send the data from it to the server.
